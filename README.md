# Fuel Price Dashboard Pipeline

This project is an automated fuel price monitoring and dashboard system that collects, processes, and visualizes fuel price data in the Philippines. The pipeline uses Python scripts and GitHub Actions to automate data collection and update the dashboard regularly.

## Features
- Automated fuel price data collection
- Data cleaning and processing
- Dashboard-ready datasets
- Fuel price trend visualization
- Automated workflow using GitHub Actions

## Tech Stack
- Python
- Pandas
- GitHub Actions
- Git & GitHub

## Purpose
The project aims to monitor fuel price trends and analyze pricing differences among fuel brands and fuel types through an automated data pipeline and dashboard visualization.
